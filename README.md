# @orkans/prettier-config `v0.0.1`

> My personal [Prettier](https://prettier.io/docs/en/configuration.html) config.

## Usage

**Install**:

```bash
$ npm i @orkans/prettier-config --save-dev
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@orkans/prettier-config"
}
```
